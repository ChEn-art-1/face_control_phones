# FaceControl - Android 面部手势辅助工具 🚀

中文版 | [English](./README.md)

**FaceControl** 是一款创新的 Android 开源应用，它利用计算机视觉技术，让用户仅通过面部动作（如眨眼、低头/抬头、摇头、张嘴）即可实现对手机的全局控制。

该项目旨在为残障人士提供一种全新的交互方式，或者在不方便手动操作（如洗手、下厨）时，让手机操作变得触手可及。

---

## ✨ 核心特性

- **实时面部追踪**：基于 **Google MediaPipe Face Landmarker**，支持 468 个面部 3D 关键点实时检测。
- **后台持续监听**：结合 **CameraX** 与 **Foreground Service**（前台服务），即使 App 退到后台或屏幕锁定，依然能持续响应。
- **全局手势模拟**：利用 **Accessibility Service**（辅助功能），无需 Root 权限即可在任何第三方应用（如抖音、浏览器）中模拟滑动、点击和长按。
- **防误触逻辑**：内置 EAR（眼径比）状态机判定，支持多种眨眼识别模式，区分生理眨眼与控制指令。

## 🛠️ 技术栈

- **语言**: Kotlin
- **UI 框架**: Jetpack Compose
- **视觉计算**: [MediaPipe Tasks Vision](https://developers.google.com/mediapipe)
- **相机支持**: CameraX (Camera2)
- **系统接口**: Android Accessibility Service API
- **后台架构**: Lifecycle-aware Foreground Service

| **低头** | 向下滑动（从顶部向底部） | 向下翻页、刷新内容 |

## 🚀 快速开始

### 环境要求
- Android 设备（运行系统 Android 8.0+ / API 26+）
- 建议使用真机测试（模拟器摄像头支持有限）

### 1. 克隆并编译

```bash
git clone https://github.com/your-id/FaceControl-Android.git
cd FaceControl-Android
# 使用 Android Studio 打开项目，Sync Gradle，然后 Run
```

### 2. 授予权限

安装后，请在系统设置中手动授予以下权限：

1. **相机权限** — 用于实时捕捉面部画面
2. **悬浮窗权限**（显示在其他应用上层）— 用于前台服务
3. **辅助功能权限** — 在"无障碍"或"辅助功能"设置中找到"FaceControl Service"并开启，允许 App 模拟全局手势

### 3. 开始使用

- 打开 App 后点击"启动人脸控制"按钮，激活摄像头和手势检测
- 通知栏会显示"FaceControl is active"表示服务运行中
- 通过面部动作即可实现免触摸的手机控制

## ⚙️ 面部动作检测算法

### 眨眼检测
利用每只眼睛 6 个关键点计算 EAR（眼径比）。当比值低于 `0.2` 时判定为闭眼。根据闭眼时长区分动作类型：
- **< 180ms** → 生理性眨眼（参与双眨眼判定）
- **180–350ms** → 忽略（防误触区间）
- **1200–2000ms**（约1.5秒）→ 触发主动长闭眼

### 低头/抬头检测
计算鼻子位置占面部总高度（额头到下巴）的比例：
- 比例 > `0.6` → 低头
- 比例 < `0.35` → 抬头

### 摇头检测
计算鼻子位置在面部水平宽度中的比例：
- 比例 > `0.75` → 向左扭头
- 比例 < `0.25` → 向右扭头

### 张嘴/闭嘴检测
利用上下嘴唇距离与嘴巴宽度的比值（MAR）判断。当比值超过 `0.5` 时判定为张嘴。
